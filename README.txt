Controlling a robot via Wifi

Overview: Real time robot control using Wifi, with obstacle avoidance, via an Android
application.

•Development of robot control functions using the L293D module.
•Use of the ESP8266 Wifi module with the Arduino board in order to setup the
server.
•Development of a GUI Android application to control the robot.
•Real-time programming using the FreeRTOS library.

Tools: C/C++ language – Arduino– FreeRTOS– Java – XML - Android Studio
